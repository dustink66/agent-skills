# Travel Agent Skills

A collection of AI agent skills for travel planning and website generation.

## Skills

### trip-planner-generator

Generates structured travel plans through interactive Q&A process.

- **Features:**
  - Interactive Q&A to collect travel requirements
  - Structured output format (daily itinerary, budget, preparation checklist, notes)
  - Child-friendly considerations
  - Practical local tips

- **Install:**
  ```bash
  npx skills add dustink66/trip-planner-generator
  ```

### trip-website-generator

Generates iOS26 liquid glass style multi-page travel websites from travel plans.

- **Features:**
  - Beautiful iOS26 liquid glass design
  - Multi-page structure (itinerary, preparation, notes, budget)
  - Interactive checklists with localStorage
  - Smooth animations
  - Mobile responsive

- **Install:**
  ```bash
  npx skills add dustink66/trip-website-generator
  ```

## Usage

1. **Generate a travel plan:**
   - Invoke `trip-planner-generator` skill
   - Answer the interactive questions
   - Get a structured travel plan

2. **Create a travel website:**
   - Invoke `trip-website-generator` skill
   - Provide the travel plan
   - Get a beautiful multi-page website
